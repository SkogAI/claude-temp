# Epic: Automated knowledge extraction pipeline

**Source**: [Github #1](https://github.com/SkogAI/claude-memory/issues/1)

## Description

## Vision

Turn claude-memory from a conversation search tool into an automated knowledge extraction pipeline. Every conversation generates structured artifacts — decisions, learnings, patterns, TODOs — processed async by local LLMs via the existing queue system.

## The Problem

Today, knowledge capture is manual:
- `decisions.md` is hand-written per session
- Learnings and patterns are lost unless someone writes them down
- Context dies when sessions end
- 1.5 years of conversation history sit
