Why for you: You had repeated issues with Claude breaking hard links and symlinks via Write operations. A pre-edit hook could check if a file is a symlink/hardlink and warn before destructive operations, preventing the most frequent source of friction.
// .claude/settings.json { "hooks": { "preEdit": { "command": "bash -c 'f=\"$CLAUDE_FILE\"; if [ -L \"$f\" ]; then echo \"WARNING: $f is a symlink pointing to $(readlink -f $f)\"; fi; links=$(stat -c %h \"$f\" 2>/dev/null || stat -f %l \"$f\" 2>/dev/null); if [ \"$links\" -gt 1 ]; then echo \"WARNING: $f has $links hard links - use in-place edit only\"; fi'" } } }
