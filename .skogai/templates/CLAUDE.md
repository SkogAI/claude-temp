# templates/ - starter templates

<what_is_this>

reusable templates for creating new content. copy-paste ready with placeholder text in <brackets>.

</what_is_this>

<available_templates>

| template           | when to use                                 |
| ------------------ | ------------------------------------------- |
| knowledge-entry.md | new learning, pattern, or general knowledge |
| project-status.md  | per-project status tracking file            |
| decision-record.md | architectural decision record (adr)         |
| task_plan.md       | planning-with-files: phases and progress    |
| findings.md        | planning-with-files: research and decisions |
| progress.md        | planning-with-files: session log and tests  |

</available_templates>

<usage>

1. copy template to destination
2. replace <bracketed_placeholders> with actual content
3. remove sections that don't apply
4. update frontmatter metadata

</usage>

<example>

```bash
cp templates/knowledge-entry.md knowledge/learnings/2026-01-19-new-insight.md
```

</example>
